Проводится оксидирование поверхности поверхности для формирования контактной маски и литография по слою окисла для вскрытия окон под основной объем транзистора.
На получившейся поверхности путем литографии формируем подзатворный диэлектрик методом осаждения, поверх которого формируем тугоплавкий металл вольфрам (W) методом осаждения.
Далее проводим литографию по слою тугоплавного металла для формирования металлического затвора.
